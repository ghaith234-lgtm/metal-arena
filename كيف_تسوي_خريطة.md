# كيف تسوي خريطة جديدة 🗺️

## الطريقة

1. روح لمجلد **`content/maps/`**
2. انسخ أي ملف موجود (مثلاً `01_arena.json`) وسمّه باسم جديد: `03_desert.json`
3. عدّل المحتوى بأي محرر نصوص (Notepad كافي)
4. ابنِ الـ APK — **الخريطة تظهر تلقائياً بشاشة اختيار الميدان!**

الترتيب بالشاشة حسب اسم الملف (لذلك نبدأهن بأرقام: `01_`, `02_`, `03_`...).

---

## الحقول الكاملة

### معلومات العرض (تظهر ببطاقة الخريطة)
| الحقل | الوصف | مثال |
|---|---|---|
| `name` | اسم الخريطة | `"مدينة الجبال"` |
| `desc` | وصف قصير | `"خريطة ضخمة فيها جبال"` |
| `tags` | مميزات | `"ضخمة • جبال • حفر"` |
| `icon` | أيقونة (إيموجي) | `"⛰"` |
| `color` | لون البطاقة [أحمر, أخضر, أزرق] من 0 لـ 1 | `[0.25, 0.6, 0.85]` |

### إعدادات الأرض
| الحقل | الوصف | الافتراضي |
|---|---|---|
| `size` | حجم الخريطة (متر × متر) | `120` |
| `ground_color` | لون الأرضية | `[0.28, 0.42, 0.2]` |
| `wall_height` | ارتفاع الجدران المحيطة | `4` |
| `dirt_patches` | عدد بقع التراب العشوائية | `0` |

### إعدادات اللعب
| الحقل | الوصف | الافتراضي |
|---|---|---|
| `spawns` | **مواقع بداية الأعداء الأربعة** `[[x, z], ...]` | — |
| `pickup_range` | نطاق توزيع صناديق الأسلحة | `48` |
| `pickup_sets` | كم مجموعة صناديق (2 = ضعف الذخيرة) | `1` |
| `nuke_spread` | نطاق إسقاط النووي حول المركز | `10` |

### الشوارع
```json
"roads_x": [-36, 0, 36],     ← شوارع عمودية عند هذي المواقع
"roads_z": [-36, 0, 36],     ← شوارع أفقية
"road_width": 8
```

### المباني
```json
"buildings": [
  [-18, -18, "tall"],    ← [x, z, النوع]
  [18, 18, "small"]
]
```
النوع: `"tall"` (برج 14م) أو `"small"` (مبنى 7م) — كلها قابلة للتدمير.

### المدن العشوائية 🏙️ (بدل الشبكة المربعة)
```json
"random_cities": [
  {
    "center": [58, 58],      ← مركز المدينة
    "radius": 42,            ← نصف قطر انتشارها
    "count": 22,             ← عدد المباني
    "tall_ratio": 0.45,      ← نسبة الأبراج (0.45 = 45% أبراج)
    "min_gap": 11,           ← أقل مسافة بين مبنيين
    "road_clear": 6,         ← مسافة الابتعاد عن الشوارع
    "rotate": 30             ← أقصى زاوية دوران عشوائية للمباني
  }
]
```

**توليد طبيعي مو مربع:**
- المباني تتوزع **بكثافة أعلى بالمركز** وتخف للأطراف (مثل المدن الحقيقية)
- كل مبنى **بزاوية عشوائية** (يكسر الشكل المستقيم)
- ذكي: **ما يحط مبنى فوق مبنى**، ولا **بنص الشارع**، ولا **فوق نقطة بداية عدو**
- تگدر تحط **عدة مدن** بنفس الخريطة (حي كبير + قرية صغيرة)

**نصيحة:** لو `count` كبير على `radius`، بعض المباني ما تنوضع (المساحة ما تكفي). وسّع `radius` أو قلّل `min_gap`.

### الأشجار والبراميل
```json
"trees": [[-8, -6], [-12, 18]],        ← [x, z]
"barrels": [[10, 20], [30, -5]],       ← براميل بمواقع محددة
"random_barrels": 12                    ← براميل عشوائية إضافية
```

### الجبال ⛰️
```json
"mountains": [
  {"pos": [-88, -55], "scale": 2.2}    ← scale: 1=صغير، 3=ضخم
]
```
مبنية من صخور متراكبة، بتصادم حقيقي — تتسلقها وتقفز منها.

### الحفر 🕳️
```json
"pits": [
  {"pos": [-35, -30], "radius": 8}     ← radius: عرض الحفرة
]
```
حفر غاطسة بالأرض بطبقات متدرجة.

### المنحدرات (للقفز) 🛹
```json
"ramps": [
  {"pos": [18, -18], "size": [8, 7], "tilt": -16, "yaw": 0}
]
```
- `tilt`: زاوية الميل (سالب = يرفعك)
- `yaw`: اتجاه المنحدر (0 / 90 / 180 / 270)

### الماء 🌊
```json
"water": [
  {"pos": [-15, 25], "size": [26, 22]}    ← [عرض, عمق]
]
```
بحيرة تبطّئ السيارة لأقل من النص وتثقل حركتها — ممتازة كمصيدة أو عائق.

### الجسور 🌉
```json
"bridges": [
  {"pos": [-15, 25], "length": 34, "width": 11, "height": 4.5, "yaw": 0}
]
```
جسر كامل: منحدر صعود + سطح مرتفع بسياج + منحدر نزول + أعمدة دعم. حطه فوق بحيرة أو حفرة!
- `yaw`: اتجاه الجسر (0 = شمال-جنوب، 90 = شرق-غرب)

### الأنفاق 🚇
```json
"tunnels": [
  {"pos": [-55, -5], "length": 26, "width": 9, "height": 4, "yaw": 90}
]
```
ممر مسقوف بجدارين وإضاءة داخلية — تمر جواه وتختفي عن الأعداء.

### الحواجز 🚧
```json
"barriers": [
  {"pos": [20, 10], "size": [8, 1], "yaw": 45}
]
```
جدران قصيرة (1.2 متر) للتغطية والاختباء — تقدر تصوّب من وراها.

### 🎨 نماذج مخصصة (props) — تنزل موديلاتك!
```json
"props": [
  {"model": "tree_pine.glb", "pos": [20, 30], "scale": 1.5, "yaw": 45, "solid": true},
  {"model": "skyscraper.glb", "pos": [-40, 10], "scale": 2.0, "y": 0, "solid": true},
  {"model": "bush.glb", "pos": [5, 5], "scale": 0.8, "solid": false}
]
```

**كيف تستخدمها:**
1. نزّل أي موديل `.glb` (شجرة، بناية، صخرة، سيارة مهجورة، أي شي!)
2. حطه بمجلد **`content/props/`**
3. اكتبه بـ `props` بملف الخريطة

| الحقل | الوصف |
|---|---|
| `model` | اسم الملف بمجلد props |
| `pos` | الموقع [x, z] |
| `y` | ارتفاع (0 = على الأرض) |
| `scale` | التكبير |
| `yaw` | الدوران بالدرجات |
| `solid` | `true` = بتصادم (تصطدم بيه) / `false` = ديكور بس |

**التصادم تلقائي** — يتولد من حجم الموديل نفسه.

### 🏔️ التضاريس (تلال ومطبات وحفر ومياه) ⭐ الأهم للجمال

```json
"terrain": {
  "enabled": true,
  "resolution": 70,
  "hills": 3.5,
  "hill_scale": 0.028,
  "bumps": 0.3,
  "craters": [
    {"pos": [-35, -30], "radius": 11, "depth": 3.0}
  ],
  "pools": [
    {"pos": [-15, 25], "radius": 15, "depth": 2.0, "level": -0.5}
  ]
}
```

| الحقل | الوصف |
|---|---|
| `enabled` | `true` = أرض بتضاريس \| `false` = أرض مسطحة |
| `resolution` | دقة التفاصيل (40 خفيف — 100 مفصّل). **70 موصى بيه** |
| `hills` | **ارتفاع التلال** (0 = مسطح، 3 = تلال متوسطة، 6 = جبلية) |
| `hill_scale` | حجم التلال (**أصغر = تلال أعرض**). 0.03 موصى بيه |
| `bumps` | **قوة المطبات الصغيرة** (0.3 = مطبات محسوسة بالقيادة) |
| `craters` | **حفر بالأرض**: `pos` الموقع، `radius` العرض، `depth` العمق |
| `pools` | **برك ماء**: نفس الحقول + `level` (مستوى سطح الماء) |

**مميزات:**
- 🚗 **القيادة تتغير كلياً** — تصعد التلال، تقفز من المطبات، تنزل الحفر
- 💧 **الماء يبطّئ سيارتك** (تلقائياً)
- 🎨 **تلوين تلقائي** — المرتفعات فاتحة، المنخفضات غامقة (عمق بصري)
- 🛣️ **الشوارع والمباني ونقاط البداية تبقى مسطحة تلقائياً** (ما تنولد بتلة)
- ✅ **كل شي يقعد على الأرض صح** — مباني، أشجار، صناديق، سيارات

**نصيحة:** ابدأ بـ `hills: 2` و `bumps: 0.25` وشوف الإحساس، بعدين زيد.

### 🌫️ الضباب (يجمّل المنظر)
```json
"fog": true,
"fog_color": [0.55, 0.62, 0.7],
"fog_density": 0.005
```
يعطي عمق للمشهد. زيد `fog_density` لجو ضبابي كثيف، أو خلي `"fog": false` لرؤية صافية.

## ⚠️ قانون الشوارع (تلقائي)

**اللعبة تمنع تلقائياً** وضع أي مبنى أو شجرة أو برميل **على الشوارع أو الأرصفة**:
- **المباني**: هامش 8م حول الشارع (لأنها عريضة ~13م)
- **الأشجار**: هامش 1.5م
- **البراميل والنماذج الصلبة**: هامش 0.8-1.5م

إذا حطيت عنصر بمكان محظور، **اللعبة تتخطاه** وتطبع تحذير بالـ Output.

**المناطق الصالحة:** بين الشوارع. مثلاً لو الشوارع عند `-36, 0, 36` وعرضها 8م، فالمباني تنحط عند **±18** أو **±52**.

## نصائح

- **الإحداثيات**: المركز (0,0)، والحدود = نصف `size` (لو size=160 فالحدود ±80)
- **خلي مسافة**: لا تحط مباني فوق الشوارع أو نقاط بداية الأعداء
- **جرّب تدريجياً**: عدّل قيمة وشغّل (F5) وشوف النتيجة
- **لو الملف فيه خطأ JSON**، اللعبة تتجاهله وتطبع تحذير بالـ Output (تأكد من الفواصل والأقواس)
- **أي حقل تحذفه** يرجع لقيمته الافتراضية — ما تحتاج تكتب كل شي
