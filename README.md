# Metal Arena — المرحلة 5 (نيترو وزلزال!) — محدّث

لعبة سيارات حربية أركيد بأسلوب Vigilante 8. هالنسخة ضافت: بوست نيترو، اصطدامات حماسية، عبوات متحركة، وعبوة عملاقة تسوي زلزال.


## تصليحات هذي النسخة

- ✅ **التحكم صار يدوي**: السيارة ما تنطلق تلقائياً — تتحرك بس لما تسحب الجويستيك (فوق=تسارع، تحت=رجوع). على الحاسبة: سهم فوق/W للتسارع.
- ✅ **أزرار اللمس تظهر صح**: انصلحت مشكلة عدم ظهور الأزرار عند بداية اللعبة (كانت تنرسم خارج الشاشة).
- ✅ **المنحدرات ما تضرّر**: الصعود على منحدر أو القفز ما يسوي ضرر بعد الآن — الضرر بس من الاصطدامات المواجهة القوية (جدار أو سيارة وجهاً لوجه)، محسوب من عمودي السطح.

## الجديد بهالنسخة

- 🔵 **نيترو (NITRO)**: بوست دفع بمخزون محدود يتعبى تلقائياً — لهب أزرق من العوادم + صوت + قوس مخزون حول الزر. والكبس بالنيترو **يوجع العدو أكثر** ويقذفه!
- 💥 **اصطدامات محسّنة**: شرر عند كل تصادم قوي + صوت خبطة + دفعة أعنف وقت البوست + ارتدادك يقل لأنك إنت الكابس
- 💣 **عبوات متحركة**: العبوة تنبض حجمها وضوءها يومض — أوضح وأخطر
- 🌋 **العبوة العملاقة (زلزال وعصف)**: زر العبوة يشتغل بطريقتين:
  - **ضغطة سريعة** = عبوة وحدة
  - **ضغط مستمر 3 ثواني** = يدمج **كل** ألغامك بعبوة عملاقة، حجمها ضخم وحولها حلقة، وانفجارها زلزال هائل + **موجة عصف مرئية** تتوسع على الأرض + اهتزاز شاشة عنيف
- 🌊 **موجة العصف**: حلقة صفراء تتوسع من كل انفجار، أكبر وأقوى بالعبوة العملاقة

## كل المزايا (تراكمي)

- فيزياء Raycast + تفاصيل سيارة (سبويلر، عوادم، جنوط، مصابيح بريك) + أصوات كاملة + تفحيط بدخان
- رشاش لا نهائي خفيف + قاذف صواريخ + صاروخ متتبع (تنهرب منه) + عبوات ناسفة
- صناديق أسلحة عشوائية بعدد محدود + نظام دم + 4 أهداف + قائمة START + اختيار شخصيات

## التشغيل

1. نزّل **Godot 4.3** (Standard، مو .NET) من godotengine.org
2. Import → اختار `project.godot` → **F5**

## التحكم

### على الموبايل (لمس) — كل الأزرار تشتغل سوية
| الزر | الوظيفة |
|---|---|
| النص الأيسر | جويستيك: **اسحب فوق=تسارع، تحت=رجوع**، يمين/يسار=توجيه |
| **FIRE** (أحمر كبير) | رشاش لا نهائي |
| **SPEC** (بنفسجي) | قاذف/متتبع — مكتوب نوعه وعدده |
| **بدّل** (رمادي صغير) | تبديل بين قاذف ومتتبع |
| **NITRO** (أزرق) | بوست دفع (شوف القوس = مخزونك) |
| **لغم** (أصفر) | ضغطة=لغم واحد، **ضغط 3 ثواني=عبوة عملاقة** (شوف حلقة الشحن) |
| **DRIFT** (برتقالي) | تفحيط |
| **BRAKE** (أزرق فاتح) | بريك / رجوع |

### على الحاسبة (للتجربة)
| المفتاح | الوظيفة |
|---|---|
| A / D أو يمين/يسار | توجيه |
| **سهم فوق / W** | تسارع للأمام |
| سهم تحت | رجوع |
| F / Enter | رشاش |
| Q | قاذف/متتبع |
| E | تبديل |
| Shift | نيترو |
| R | لغم (استمر بالضغط 3 ثواني للعملاقة) |
| Space | تفحيط |
| S | بريك / رجوع |

## ضبط الإحساس

- القيادة والرشاش والنيترو: أعلى `scripts/car.gd` (`boost_force`، `boost_drain`، `boost_ram_damage`، `mine_hold_time`...)
- قوة العبوة العملاقة: دالة `_plant_mega_mine` بـ `car.gd` — تكبر مع عدد الألغام المدموجة
- الانفجار والعصف والزلزال: `scripts/fx.gd` (بارامتر `strength`)
- كمية الصناديق: `scripts/pickup.gd` (`AMOUNTS`)
