# Metal Arena — المرحلة 12 (أزرار أسلحة منفصلة)

لعبة سيارات حربية أركيد بأسلوب Vigilante 8. هالنسخة: وضع ضد الذكاء الاصطناعي، هزاز الهاتف، درع مؤقت، وتصليح.

## الجديد بهالنسخة

- 🏢 **مباني أكبر**: المباني صارت ضعف الحجم (أطول وأعرض) — تبين أوضح وتعطي إحساس مدينة حقيقية
- 📦 **ذخيرة أكثر بالصناديق**: القاذف 5 (بدل 2)، المتتبع 3 (بدل 1)، اللغم 4 (بدل 2)
- 🎯 **زر القاذف منفصل عن المتتبع**: صار لكل سلاح زر مستقل (قاذف / متتبع) — خلصنا من زر "بدّل"، تضغط اللي تريده مباشرة
- 🚀 **ايم اسست للقاذف**: صاروخ القاذف صار يميل نحو أقرب عدو تلقائياً مثل الرشاش — تصويب أسهل بكثير

## المزايا السابقة ☢️

**حدث السلاح النووي** — أكبر حدث بالخريطة، كل 5 دقائق:

1. 📢 **إنذار للكل**: صافرة + رسالة "السلاح النووي قادم!"
2. 🚁 **هليكوبتر** تطير وتنزل **صندوق محصّن** (300 صحة) وسط الخريطة
3. ⚔️ **صراع على الصندوق**: الكل يضربه حتى ينكسر ويطلع السلاح
4. 💚 **السلاح النووي** يطلع (كرة خضراء مضيئة بعمود ضوء يبين من بعيد + نقطة صفراء بالرادار)
5. 🏃 **أول سيارة تلمسه تاخذه** — وعندها **10 ثواني** للإطلاق
6. 🎯 **الباقي يطاردها**: لازم يدمرون الحامل قبل الإطلاق! لو دمروه (أو دخل الحالة الحرجة)، السلاح **يطيح ويرجع الصراع**
7. 🚀 **لو أطلق**: صاروخ يصعد للسماء وينزل → **انفجار هائل يغطي نص الخريطة** + **سلو موشن** + **وميض أبيض قوي** + اهتزاز عنيف

**الأعداء الأذكياء يتصارعون عليه بعد** — ياخذونه، يهربون فيه، ويطلقونه عليك! وإذا إنت أخذته، يطاردونك بكل قوة.

## كل شي (تراكمي)

- سيارة أنيقة + مدينة + رادار + أعداء أذكياء + درع + تصليح + طقس + الحالة الحرجة
- رشاش + قاذف + صاروخ متتبع + عبوات + عبوة عملاقة (زلزال) + انفجارات + أصوات
- قائمة START + اختيار شخصيات بمعاينة 3D

## التشغيل

1. نزّل **Godot 4.3** (Standard، مو .NET) من godotengine.org
2. Import → اختار `project.godot` → **F5**

## التحكم (لمس)

جويستيك يسار (فوق=تسارع)، وأزرار يمين: FIRE (رشاش)، SPEC (قاذف/متتبع)، بدّل، NITRO، لغم (ضغط 3 ثواني=عملاقة)، DRIFT، BRAKE. كلها تشتغل سوية.

حاسبة: WASD/أسهم، F رشاش، Q خاص، E تبديل، Shift نيترو، R لغم، Space تفحيط، **X تفجير** (بالحالة الحرجة).

## الصناديق بالخريطة

| الصندوق | الشكل | الأثر |
|---|---|---|
| قاذف | مكعب برتقالي | 2 صاروخ مستقيم |
| متتبع | مكعب أحمر | 1 صاروخ يلحق |
| لغم | مكعب أصفر | 2 عبوة |
| **درع** | كرة زرقا بحلقات | فقاعة تمتص الضرر 6 ثواني |
| **تصليح** | علامة + خضرا | +20% صحة |

## ضبط الإعدادات

- الذكاء الاصطناعي: `scripts/car_ai.gd` (مدى الرؤية، مسافة الالتفاف، متى يهرب)
- الدرع والتصليح: `scripts/pickup.gd` (`SHIELD_SECONDS`، `REPAIR_FRACTION`)
- التوجيه الذكي: أعلى `scripts/car.gd` (`aim_assist_angle`، `aim_assist_strength`)
- الهزاز: دالة `vibrate` بـ `scripts/fx.gd`
