# Metal Arena — المرحلة 4 (أسلحة وأصوات!)

لعبة سيارات حربية أركيد بأسلوب Vigilante 8. هالنسخة: منظومة أسلحة كاملة + أصوات + تفحيط + صناديق عشوائية.

## شنو بيها

- 🚗 فيزياء Raycast suspension بدرفت وقفزات واستعادة تلقائية عند الانقلاب
- 🎨 تفاصيل سيارة: سبويلر، صادم، عوادم، جنوط، مصابيح أمامية مضيئة، **مصابيح خلفية تشتد مع البريك**، ميلان جسم مع الانعطاف
- 🔊 **أصوات كاملة**: هدير محرك يتغير مع السرعة، صرير تفحيط، طلقات، انفجارات، صافرة صاروخ متتبع (كلها مولّدة رياضياً — بلا ملفات خارجية)
- 🏎️ **تفحيط**: دخان من الإطارين الخلفيين + رسالة "تفحيط X ثانية!" إذا كملت أكثر من ثانية
- 🔫 رشاش لا نهائي بس خفيف + ارتداد ووميض فوهة
- 🚀 **قاذف صواريخ** مستقيم (35 ضرر، انفجار منطقة)
- 🎯 **صاروخ متتبع** يلاحق أقرب عدو — دورانه محدود فتگدر تهرب منه بدرفت حاد أو تختفي ورا عمود، ويصفّر وهو جاي!
- 💣 **عبوة ناسفة** تنزرع وراك، تومض بالأحمر، وانفجارها **يطيّر السيارة بالهوا**
- 📦 صناديق أسلحة موزعة **عشوائياً** بالخريطة (عائمة تدور)، ترجع بمكان جديد كل 12 ثانية — والأسلحة الخاصة **بعدد محدود**
- 💥 انفجارات بضرر منطقة ودفعة تقلب السيارات + اهتزاز كاميرا حسب القرب
- ❤️ نظام دم كامل + 4 سيارات أهداف تتفجر وترجع + عداد KILLS
- 🖥️ HUD: سرعة، شريط صحة، ذخيرة الأسلحة (والمختار بين « »)، علامة إصابة

## التشغيل

1. نزّل **Godot 4.3** (Standard، مو .NET) من godotengine.org
2. Import → اختار `project.godot` → **F5**

## التحكم

### على الموبايل (لمس)
| العنصر | الوظيفة |
|---|---|
| النص الأيسر من الشاشة | جويستيك توجيه يظهر وين ما تلمس |
| **FIRE** الأحمر الكبير | رشاش (لا نهائي) |
| **SPEC** البنفسجي (فوق FIRE) | السلاح الخاص — مكتوب عليه نوعه وعدده (R/H/M) |
| **بدّل** الصغير (فوق SPEC) | تبديل السلاح الخاص |
| **DRIFT** البرتقالي | تفحيط |
| **BRAKE** الأزرق | بريك / رجوع |
| — | التسارع تلقائي، وكل الأزرار تشتغل سوية (متعدد الأصابع) |

### على الحاسبة (للتجربة)
| المفتاح | الوظيفة |
|---|---|
| أسهم أو A/D | توجيه |
| F أو Enter | رشاش |
| Q | السلاح الخاص |
| E | تبديل السلاح |
| Space | تفحيط |
| S أو سهم أسفل | بريك / رجوع |

## القائمة والشخصيات

اللعبة تبدي بشاشة **START GAME** → شاشة **اختيار المقاتل** (ستايل ببجي): صورة الشخصية، سيارتها تدور 3D، وأشرطة مواصفات تأثر فعلياً باللعب.

**حتى تضيف شخصية جديدة:**
1. حط صورتها بمجلد `assets/` (مربعة يفضل)
2. افتح `scripts/global.gd` وضيف قاموس جديد بقائمة `CHARACTERS`
3. خلص! تظهر بشاشة الاختيار تلقائياً

## ضبط الإحساس (Tuning)

قيم القيادة والرشاش بأعلى `scripts/car.gd`. قيم الأسلحة الخاصة: الصواريخ بدالة `_launch_projectile` بنفس الملف، اللغم بـ `scripts/mine.gd`، كميات الصناديق بـ `scripts/pickup.gd` (قاموس `AMOUNTS`).

## التحديث على GitHub — سهل جداً

**أي ملف يتغير → ترفعه → APK جديد يطلع بروحه.**
Add file → Upload files → اسحب الملفات → Commit → Actions يبني → Artifacts فيها الـ APK. للتفاصيل شوف `GITHUB_BUILD.md`.
