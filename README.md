# Metal Arena — المرحلة 6 (مدينة قابلة للتدمير!)

لعبة سيارات حربية أركيد بأسلوب Vigilante 8. هالنسخة: سيارة أجمل، مدينة بشوارع ومباني تتدمر، ورادار يبين الأعداء.

## الجديد بهالنسخة

- 🚗 **سيارة أجمل بالكامل**: هيكل متعدد الطبقات، مقدمة مدببة، كابينة منحدرة بزجاج، سقف، قوس حماية، تنانير جانبية، أقواس عجلات، صادم بقضبان، سبويلر مائل، عوادم رباعية، ومصابيح — شكل سيارة قتال حقيقية
- 🏙️ **مدينة**: أرض حشائش خضراء + شبكة شوارع رمادية بخطوط صفراء + أحياء
- 🏢 **مباني تتدمر**: بنايات عالية (220 صحة) وصغيرة (120 صحة) بنوافذ مضيئة — تنهار وتتفتت حطام
- 🌳 **أشجار تتدمر** (35 صحة): جذع + أوراق، تتكسر وتطير حطام أخضر
- 🛢️ **براميل متفجرة** (18 صحة): تنفجر انفجار كبير لما تضربها أو تصدمها — سلاح بيئي!
- 🎯 **رادار (خريطة مصغرة)**: فوق يمين، يبين موقعك (سهم أزرق) والأعداء (نقاط حمرا)، يدور مع اتجاهك، والأعداء البعيدين يبينون على حافة الرادار
- ✅ **تصليح القاذف والصاروخ المتتبع**: صاروا يأثرون على السيارات صح — انصلحت مشكلة عدم إحداث الضرر (كانوا ينفجرون بمطلقهم فوراً)

## كل شي بالتفصيل (تراكمي)

- فيزياء + تحكم يدوي بالجويستيك (فوق=تسارع) + تفحيط + نيترو
- رشاش لا نهائي + قاذف + صاروخ متتبع + عبوات ناسفة + عبوة عملاقة (زلزال)
- صناديق أسلحة عشوائية + نظام دم + انفجارات + اهتزاز شاشة
- قائمة START + اختيار شخصيات + أصوات كاملة

## التشغيل

1. نزّل **Godot 4.3** (Standard، مو .NET) من godotengine.org
2. Import → اختار `project.godot` → **F5**

## التحكم (لمس)

| الزر | الوظيفة |
|---|---|
| النص الأيسر | جويستيك: **فوق=تسارع، تحت=رجوع**، يمين/يسار=توجيه |
| **FIRE** | رشاش لا نهائي (يدمّر المباني والأشجار بعد) |
| **SPEC** | قاذف/متتبع |
| **بدّل** | تبديل السلاح الخاص |
| **NITRO** | بوست دفع |
| **لغم** | ضغطة=واحد، ضغط 3 ثواني=عبوة عملاقة |
| **DRIFT** / **BRAKE** | تفحيط / بريك |

على الحاسبة: أسهم/WASD قيادة، F رشاش، Q خاص، E تبديل، Shift نيترو، R لغم، Space تفحيط.

## صحة العناصر (كم ضربة تتحمل)

| العنصر | الصحة |
|---|---|
| برميل متفجر | 18 (وينفجر!) |
| شجرة | 35 |
| بناية صغيرة | 120 |
| بناية عالية | 220 |

القيم بدالة `setup()` بـ `scripts/destructible.gd` — غيّرها مثل ما تحب.
