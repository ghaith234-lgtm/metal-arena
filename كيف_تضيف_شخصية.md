# كيف تضيف شخصية (سيارة) جديدة 🚗

## الفكرة الجديدة (الأبسط والأضمن)

1. نزّل أي سيارة من الإنترنت
2. افتحها بـ **Blender** واحذف عجلاتها (اضغط على كل عجلة → Delete) — **إلزامي**: ما أكو أي دمج تلقائي، عجلات اللعبة هي الوحيدة
3. صدّرها **glb**: File → Export → glTF 2.0 → صيغة **glTF Binary (.glb)**
4. حطها بمجلد الشخصية — **اللعبة تحط عجلاتها هي وتضبطها تلقائياً على أبعاد الجسم!**

عجلات اللعبة تدور وتنعطف وتتبع الأرض بالتعليق — لأي جسم مهما كانت أبعاده.

## هيكلية المجلد

```
content/characters/
  اسم-الشخصية/          ← اسم المجلد = اسم الشخصية باللعبة
    car.glb              ← جسم السيارة (بدون عجلات)
    portrait.png         ← صورة الشخصية
    stats.json           ← الإعدادات
```

## ملف stats.json — المرجع الكامل

```json
{
  "class": "medium",
  "subtitle": "وصف يظهر تحت الاسم",
  "scale": 1.0,
  "rotate_y": 0,
  "lift": 0.0,
  "wheels": {
    "front": 0.85,
    "back": 0.85,
    "width": 0.62,
    "height": -0.15,
    "size": 1.0
  }
}
```

| الحقل | الوصف |
|---|---|
| `class` | فئة السيارة: `"tank"` (دبابة: كبيرة/بطيئة/قوية)، `"medium"` (متوازنة)، `"fast"` (سريعة/صغيرة/ضعيفة) |
| `scale` | **تكبير السيارة كاملة** (جسم + عجلات + مسافات سوا): 1.0 طبيعي، 1.3 أكبر 30%، 0.8 أصغر — يحل اختلاف أحجام الموديلات بين المواقع |
| `rotate_y` | تدوير الجسم بالدرجات لو طالع باتجاه غلط (90 / 180 / -90) |
| `flip` | **`true` لو السيارة مقدمتها بالخلف** — يقلبها 180 درجة (أسهل من rotate_y) |
| `lift` | **نزول وصعود الجسم عن العجلات**: موجب يرفعه، سالب ينزّله (جرّب 0.1 أو -0.1) |
| `wheels` | ضبط عجلات اللعبة — **احذف الحقل كله وتنضبط تلقائياً** على أبعاد الجسم |
| `wheels.front` | بعد العجلات الأمامية عن مركز السيارة |
| `wheels.back` | بعد العجلات الخلفية عن المركز |
| `wheels.width` | نصف المسافة بين عجلات اليمين واليسار |
| `wheels.height` | ارتفاع نقطة تعليق العجلات (الافتراضي -0.15) |
| `wheels.size` | حجم العجلات (1.0 عادي، 1.5 أكبر، 0.7 أصغر) |
| `hide_game_wheels` | `true` لو الموديل عجلاته منحوتة بالجسم وتريد إخفاء عجلات اللعبة |

**الشاصي التلقائي:** اللعبة تولّد شاصي رابط (محورين + عمود فقري + قوائم تعليق) بين العجلات والجسم، يتكيف تلقائياً مع أبعاد العجلات — فما تبين السيارة "طايفة" فوق عجلاتها.

**نصيحة:** ابدأ بدون حقل `wheels` نهائياً (الضبط التلقائي زين بمعظم الحالات)، وإذا العجلات مو مضبوطة على جسمك، ضيف الحقل وعدّل قيمة قيمة.

## 🔫 ملف weapons.json — ارتفاعات الأسلحة (اختياري)

ملف منفصل داخل مجلد الشخصية يضبط **صعود ونزول** منصات الأسلحة الثلاثة حتى تقعد صح على جسمك (والطلق يطلع من مكانها الجديد تلقائياً):

```json
{
  "gun": 0.36,
  "rocket": 1.12,
  "homing": 0.52
}
```

| الحقل | السلاح | القيمة الافتراضية |
|---|---|---|
| `gun` | سبطانة الرشاش (أمام) | 0.36 |
| `rocket` | القاذف (فوق) | 1.12 |
| `homing` | المتتبع (يمين) | 0.52 |

زيد الرقم = السلاح يصعد، نقّصه = ينزل. بدون الملف = القيم الافتراضية.

## ⚠️ التكستشرات الخارجية (Kenney وغيرها)

بعض ملفات glb تعتمد على مجلد `Textures/` جنبها. **انسخه معاها** داخل مجلد الشخصية:
```
content/characters/الوحش/
  car.glb
  Textures/colormap.png    ← لو موجود بمجلد التنزيل
```

## من وين تنزل سيارات؟

- **kenney.nl** → Car Kit (مجاني CC0 — الأفضل)
- **poly.pizza** — فلتر سيارات، نزّل glTF (.glb)
- **quaternius.com** → Ultimate Vehicles
- لو الملف .fbx أو .obj → حوّله ببلندر (Import ثم Export glb) أو موقع تحويل أونلاين

**المواصفات:** أقل من 10 آلاف مضلع، رخصة CC0/CC-BY، بدون أنيميشن.
